Программа предназначена для построения базиса Грёбнера для заданной системы уравнений.

Для Вас доступны следующие команды:
	
	input(или in) - ввод исходной системы уравнений
	output(или out) - вывод исходной системы уравнений
	basis - формирование базиса Грёбнера для введенной системы
	outputbasis(или outb) - вывод базиса Грёбнера
	decision(или dec) - свести исходные уравнения к уравнению с одной неизвестной
	
	cls, clear - очистить экран
	quit, q, exit - выход
